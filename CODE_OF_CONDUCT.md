# Code of Conduct

This project expects respectful, constructive participation.

## Expected behavior

- Be considerate of different experience levels and backgrounds.
- Discuss ideas, tradeoffs, and code without personal attacks.
- Keep feedback specific, actionable, and relevant to the project.
- Respect maintainers' decisions about scope and priorities.

## Unacceptable behavior

- Harassment, threats, or discriminatory language.
- Personal insults or repeated hostile comments.
- Publishing private information without permission.
- Disruptive behavior that makes collaboration harder.

## Reporting

Report conduct concerns through the repository security or maintainer contact
channels. Maintainers may remove comments, close discussions, or block
participants when needed to keep the project usable.
